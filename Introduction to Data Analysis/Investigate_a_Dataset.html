Introduction¶

The ultimate Soccer database for data analysis and machine learning

What is Given: 25,000 matches +10,000 players 11 European Countries with their lead championship Seasons 2008 to 2016 Players and Teams' attributes* sourced from EA Sports' FIFA video game series, including the weekly updates Team line up with squad formation (X, Y coordinates) Betting odds from up to 10 providers Detailed match events (goal types, possession, corner, cross, fouls, cards etc...) for +10,000 matches

We are going to analyse the soccer database and come up with some meaningful insights.

At the end of the analysis we will answer these questions.

1. Who was the best player of the year 2010?
2. How many total home and away goals were scored in each league?
3. Give a plot of ranks of top 10 players for the last 5 year period

In [11]:
# To accomplish this we need to run the above piece of code 5 times 
# To make the code concise, lets create a function instead of writing the same code over and over

def top_N_players(df,year,n=10):
    """Function which return N Number of Top Player at end of Year"""
    df_top = players_list[players_list['date'].dt.year == year]
    df_top = df_top.sort_values('date').groupby('player_api_id').last() 
    df_top = df_top.sort_values(['overall_rating','potential']).tail(n)
    df_top = df_top.sort_values(['overall_rating','potential'],ascending=False)
    return df_top

Limitations in Data¶

To be frank, there are a couple of limitations to this dataset, which may/may not cause some problems if we happpen to analyse the data set further.

You will notice that some players are missing from the lineup (NULL values). There are also NULL values in the ranks. This made is difficult to plot the ranks of players. We had to work around it by raplacing null with some random value. This will not be pretty in the long run.
